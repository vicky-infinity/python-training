Q21. Company Payroll System
--------------------------------------------------

Create a base class Employee.

Attributes:
- employee_id
- name
- basic_salary

Methods:
- calculate_salary()

Create two child classes:

- FullTimeEmployee
- ContractEmployee

Each class should calculate the final salary differently.

Tasks:

- Create a list containing both types of employees.
- Use polymorphism to calculate the salary of every employee.
- Build a dictionary {employee_name: final_salary} for employees earning more than ₹60,000.
- Print the dictionary.
