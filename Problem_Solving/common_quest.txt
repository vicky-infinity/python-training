Q1. Student Grades
Create a class Student with:

- attributes: name, marks
- method is_pass() → returns True if marks ≥ 40

Q2. Bank Account
Create a class BankAccount:

- attributes: account_holder, balance

Print final balance.


Q3. Rectangle

Q4. Product Discount (Simple)

Q5. Employee Bonus
