Q31. Employee Attendance Logger
--------------------------------------------------

Create a class Employee.

Attributes:
- employee_id
- name
- attendance

Methods:
- mark_attendance(status)

Requirements:

- Create a decorator to log the execution time of mark_attendance().
- Log every attendance update using the logging module.
- Raise an exception for invalid attendance status.
- Create 6 employees and update their attendance.
- Print all employee records.

Q32. Banking Transaction Audit
--------------------------------------------------

Create a class BankAccount.

Attributes:
- account_number
- account_holder
- balance

Methods:
- deposit(amount)
- withdraw(amount)

Requirements:

- Log every successful transaction.
- Raise appropriate exceptions for invalid deposits or withdrawals.
- Create a decorator to print a success message after every valid transaction.
- Create 5 bank accounts.
- Print the final balance of every account.
